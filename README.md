# Reproducible Pipeline for NLSE vs LSE Reservoir Experiments

# Reproducible Pipeline Overview

This repository provides two fully reproducible paths for generating all figures and statistical results associated with the manuscript.

----------------------------------

## A. Canonical (Preload) Pipeline (fast)

This path uses preloaded `.npz` files that correspond exactly to the figures and statistics in the paper.  
No new NLSE/LSE simulations are generated.

Run the scripts **in this order**:

1. `2p_preload_Masterscript_analysis.py`  
2. `3p_preload_1B_fig2_generator.py`  
3. `6p_preload_etaV2paper1B_viz_final.py`  
4. `7p_preload_G_T_derivatives.py`  
5. `8p_preload_statistical_analysis.py`

This reproduces all figures and quantitative results deterministically.

## B. Full Pipeline — regenerate all data from scratch

This path recomputes all simulations, sweeps, and statistical analyses.

Run the scripts in the following order:

1. `1_Master_script_statistics.py` — 15-point η robustness sweep (10 seeds each)  
2. `2_Masterscript_analysis.py` — decay-rate regression  
3. `3_1B_fig2_generator.py` — superiority-margin figure  
4. `5_etaV2paper1B_final.py` — time sweep (T) and nonlinearity sweep (g)  
5. `6_etaV2paper1B_viz_final.py` — g-margin visualization  
6. `7_G_T_derivatives.py` — derivative / curvature analysis  
7. `8_statistical_analysis.py` — t-tests, Cohen’s d, Bonferroni corrections  

## ⚠️ Computation Time
- **Scripts 1 and 5 are computationally expensive** (runtime: hours, depending on hardware).  
- All other scripts run quickly.

---

# *NOTES:*
- Canonical `.npz` files in `preload_data/` reproduce **exactly** the figures and statistics in the manuscript.  
- Appendix parameter sweeps (T-sweep and g-sweep) include stochastic jitter; regenerated results may show **small numeric variation**.  
- All script paths are **relative to the repository root**.  
- The directories `figures/` and `preload_figures/` will be created and populated automatically during execution.

## Optional: Quickstart Demo (Not Used in the Paper)
For users who want a short, fast, educational demonstration of the NLSE vs LSE robustness effect, a reduced-parameter Quickstart script is included:

### Script Location:
`demos/quickstart_demo.py`

### Overview:

This script:

- Runs in ~10–15 minutes (compared to hours for the full pipeline)
- Uses fewer seeds, fewer η values, and smaller datasets
- Reproduces the qualitative behavior of NLSE vs LSE decay rates
- Is intended only for illustration and sanity-checking, **not for reproduction** of the paper’s figures or statistics

### Important Note:
The official results in the manuscript come exclusively from the canonical `.npz` datasets or the full regeneration pipeline described above.
